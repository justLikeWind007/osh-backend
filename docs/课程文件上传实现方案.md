# 课程管理系统文件上传实现方案

## 一、需求概述

### 核心原则
1. **职责分离**:课程创建仅保存基础信息，不处理任何文件上传
2. **独立接口**:文件上传通过独立的上传接口在课程创建后执行
3. **等待集成**:调用文件模块提供的统一上传接口 `upload(MultipartFile, UploadPathEnum)`
4. **事务一致**:上传失败时保证数据一致性

---

## 二、数据库表结构分析

### 现有表字段

#### 1. osh_coures（课程主表）
- `cover` VARCHAR - 存储课程封面图片 URL（一对一）
- `tryContent` TEXT - 可存储试看文档 URL（一对一）

#### 2. osh_course_section（章节表）
- `mediaUrl` VARCHAR - 存储课时视频 URL（一对多）
- `cover` VARCHAR - 章节封面 URL（可选）

#### 3. osh_course_material（资料表）
- `fileUrl` VARCHAR - 存储资料文件 URL（一对多）
- `fileType` VARCHAR - 文件类型
- `fileSize` BIGINT - 文件大小

### 结论
**无需新建关联表**,现有表结构已满足需求:
- 课程封面 → `osh_coures.cover`
- 课时视频 → `osh_course_section.mediaUrl`
- 课程资料 → `osh_course_material.fileUrl`

## 三、文件上传架构设计

### 3.1 接口分类

```
课程文件上传
├── 课程封面上传 (Cover Upload)
│   └── POST /pc/course/cover/upload
├── 课时视频上传 (Video Upload)
│   └── POST /pc/course/section/video/upload
└── 课程资料上传 (Material Upload)
    └── POST /pc/course/material/upload
```

### 3.2 技术栈

```java
// 等待文件模块提供的接口
FileUploadUtils.upload(MultipartFile file, UploadPathEnum pathEnum)

// 课程模块调用方式
String fileUrl = FileUploadUtils.upload(file, UploadPathEnum.COURSE_COVER);
String fileUrl = FileUploadUtils.upload(file, UploadPathEnum.COURSE_VIDEO);
String fileUrl = FileUploadUtils.upload(file, UploadPathEnum.COURSE_MATERIAL);
```

### 3.3 文件类型规划（需与文件模块确认）

| 枚举项 | 用途 | 允许扩展名 | 大小限制 | 存储路径 |
|--------|------|-----------|---------|---------|
| COURSE_COVER | 课程封面 | bmp, gif, jpg, jpeg, png | 5MB | course/cover/ |
| COURSE_VIDEO | 课时视频 | mp4, avi, mov, mkv, wmv, flv | 500MB | course/video/ |
| COURSE_MATERIAL | 课程资料 | zip, rar, pdf, doc, docx, ppt, pptx | 100MB | course/material/ |

## 四、详细实现方案

### 4.1 课程封面上传接口

#### Controller 层
```java
/**
 * 上传课程封面图片
 */
@Anonymous
@Log(title = "课程封面", businessType = BusinessType.UPLOAD)
@ApiOperation("上传课程封面")
@PostMapping("/cover/upload")
public R<Void> uploadCourseCover(
        @RequestParam("file") MultipartFile file,
        @RequestParam("courseId") Long courseId)
{
    try {
        Long userId = getUserId();
        courseManageService.uploadCourseCover(file, courseId, userId);
        return R.ok();
    } catch (FileSizeLimitExceededException e) {
        log.error("上传封面失败：文件大小超限", e);
        return R.fail("封面图片大小不能超过 5MB");
    } catch (InvalidExtensionException e) {
        log.error("上传封面失败：文件格式不支持", e);
        return R.fail("封面图片仅支持 bmp、gif、jpg、jpeg、png 格式");
    } catch (Exception e) {
        log.error("上传封面失败：{}", e.getMessage(), e);
        return R.fail("上传失败：" + e.getMessage());
    }
}
```

#### Service 层
```java
/**
 * 上传课程封面
 * 实现效果：将封面图片上传到文件服务器，并更新课程表的 cover 字段
 * 
 * @param file 封面文件
 * @param courseId 课程 ID
 * @param userId 用户 ID
 */
@Override
@Transactional(rollbackFor = Exception.class)
public void uploadCourseCover(MultipartFile file, Long courseId, Long userId) {
    // 1. 校验课程是否存在
    OshCoures course = oshCouresMapper.selectCourseById(courseId);
    if (course == null) {
        throw new ServiceException("课程不存在");
    }
    
    // 2. 校验权限（仅课程创建者或管理员可上传）
    if (!course.getCreateBy().equals(String.valueOf(userId))) {
        throw new ServiceException("无权限上传该课程的封面");
    }
    
    // 3. 调用文件上传接口（等待文件模块提供）
    String coverUrl = FileUploadUtils.upload(file, UploadPathEnum.COURSE_COVER);
    
    // 4. 更新课程封面 URL
    OshCoures updateCourse = new OshCoures();
    updateCourse.setId(courseId);
    updateCourse.setCover(coverUrl);
    updateCourse.setUpdateTime(DateUtils.getNowDate());
    updateCourse.setUpdateBy(String.valueOf(userId));
    
    int result = oshCouresMapper.updateCourse(updateCourse);
    if (result <= 0) {
        throw new ServiceException("更新课程封面失败");
    }
}
```

### 4.2 课时视频上传接口

#### Controller 层
```java
/**
 * 上传课时视频

@PreAuthorize("@ss.hasPermi('system:course:video:upload')")

    } catch (InvalidExtensionException e) {

    } catch (Exception e) {
        log.error("上传视频失败：{}", e.getMessage(), e);
        return R.fail("上传失败：" + e.getMessage());
    }
}
```

#### Service 层
```java
/**
 * 上传课时视频
 * 实现效果：上传视频文件并更新章节表的 mediaUrl 字段
 * 
 * @param file 视频文件
 * @param courseId 课程 ID
 * @param sectionId 章节 ID
 * @param userId 用户 ID
 * @return 视频上传结果 VO
 */
@Override
@Transactional(rollbackFor = Exception.class)
public VideoUploadVO uploadSectionVideo(MultipartFile file, Long courseId, Long sectionId, Long userId) {
    // 1. 校验章节是否存在且属于该课程
    OshCourseSection section = courseSectionMapper.selectSectionById(sectionId);
    if (section == null || !section.getCourseId().equals(courseId)) {
        throw new ServiceException("章节不存在或不属于该课程");
    }
    
    // 2. 调用文件上传接口
    String videoUrl = FileUploadUtils.upload(file, UploadPathEnum.COURSE_VIDEO);
    
    // 3. TODO: 提取视频元数据（时长、分辨率等）
    // VideoMetadata metadata = videoService.extractMetadata(file);
    
    // 4. 更新章节的视频 URL
    OshCourseSection updateSection = new OshCourseSection();
    updateSection.setId(sectionId);
    updateSection.setMediaUrl(videoUrl);
    // updateSection.setDuration(metadata.getDuration()); // 后续实现
    // updateSection.setVideoResolution(metadata.getResolution()); // 后续实现
    updateSection.setUpdateTime(DateUtils.getNowDate());
    updateSection.setUpdateBy(String.valueOf(userId));
    
    int result = courseSectionMapper.updateSection(updateSection);
    if (result <= 0) {
        throw new ServiceException("更新章节视频失败");
    }
    
    // 5. 构建返回结果
    VideoUploadVO vo = new VideoUploadVO();
    vo.setVideoUrl(videoUrl);
    vo.setSectionId(sectionId);
    // vo.setDuration(metadata.getDuration()); // 后续实现
    
    return vo;
}
```

### 4.3 课程资料上传接口

#### Controller 层

```java
/**
 * 上传课程资料
 */
@PreAuthorize("@ss.hasPermi('system:course:material:upload')")
@Log(title = "课程资料", businessType = BusinessType.UPLOAD)

        @RequestParam("file") MultipartFile file,

    } catch (FileSizeLimitExceededException e) {

    } catch (InvalidExtensionException e) {
        log.error("上传资料失败：文件格式不支持", e);
        return R.fail("资料格式不支持");
    } catch (Exception e) {
        log.error("上传资料失败：{}", e.getMessage(), e);
        return R.fail("上传失败：" + e.getMessage());
    }
}
```

#### Service 层
```java
/**
 * 上传课程资料
 * 实现效果：上传资料文件并在 osh_course_material 表中插入记录
 * 
 * @param file 资料文件
 * @param courseId 课程 ID
 * @param materialName 资料名称
 * @param userId 用户 ID
 * @return 资料 ID
 */
@Override
@Transactional(rollbackFor = Exception.class)
public Long uploadCourseMaterial(MultipartFile file, Long courseId, String materialName, Long userId) {
    // 1. 校验课程是否存在
    OshCoures course = oshCouresMapper.selectCourseById(courseId);
    if (course == null) {
        throw new ServiceException("课程不存在");
    }
    
    // 2. 调用文件上传接口
    String fileUrl = FileUploadUtils.upload(file, UploadPathEnum.COURSE_MATERIAL);
    
    // 3. 获取文件信息
    String fileName = file.getOriginalFilename();
    String fileType = "";
    if (fileName != null && fileName.contains(".")) {
        fileType = fileName.substring(fileName.lastIndexOf(".") + 1).toLowerCase();
    }
    
    // 4. 插入资料记录
    Map<String, Object> params = new HashMap<>();
    params.put("courseId", courseId);
    params.put("materialName", StringUtils.defaultIfEmpty(materialName, fileName));
    params.put("fileUrl", fileUrl);
    params.put("fileType", fileType);
    params.put("fileSize", file.getSize());
    params.put("isDownloadable", 1);
    params.put("sortOrder", 0);
    params.put("createBy", String.valueOf(userId));
    params.put("createTime", DateUtils.getNowDate());
    
    materialMapper.insertMaterial(params);
    
    // 5. 返回资料 ID
    return (Long) params.get("id");
}
```

## 五、异常处理机制

### 5.1 文件上传相关异常

| 异常类型 | 触发条件 | 用户提示 |
|---------|---------|---------|
| `FileSizeLimitExceededException` | 文件超出枚举定义的大小限制 | "{文件类型} 大小不能超过 {X}MB" |
| `FileNameLengthLimitExceededException` | 文件名超出长度限制 | "文件名过长，请重命名" |
| `InvalidExtensionException` | 文件格式不在允许列表中 | "仅支持 {格式列表} 格式" |
| `IOException` | 文件读写失败 | "上传失败：IO 异常" |
| `FileUploadException` | 其他上传异常 | "上传失败：[具体原因]" |

### 5.2 业务异常

| 异常场景 | 异常类型 | 用户提示 |
|---------|---------|---------|
| 课程不存在 | `ServiceException` | "课程不存在" |
| 章节不存在 | `ServiceException` | "章节不存在" |
| 无权限操作 | `ServiceException` | "无权限上传该课程的文件" |
| 数据库更新失败 | `ServiceException` | "更新失败" |

### 5.3 事务回滚策略

```java
@Transactional(rollbackFor = Exception.class)
public void uploadCourseCover(...) {
    // 1. 文件上传失败 → 自动回滚
    String url = FileUploadUtils.upload(...);
    
    // 2. 数据库更新失败 → 自动回滚
    oshCouresMapper.updateCourse(...);
    
    // 3. 任何未捕获的异常 → 自动回滚
}
```

## 六、使用流程示例

### 6.1 完整课程创建流程

```
步骤 1: 创建课程（不含文件）
POST /pc/course/create
Body: {
    "title": "Java 高级编程",
    "type": "media",
    "price": 99.00,
    ...
}
Response: { "code": 200, "data": 123 }  // 返回课程 ID

步骤 2: 上传课程封面
POST /pc/course/cover/upload?courseId=123
FormData: file=[封面图片]
Response: { "code": 200 }

步骤 3: 创建章节
POST /pc/course/section
Body: {
    "courseId": 123,
    "title": "第一章 Java 基础",
    "sectionType": "chapter"
}
Response: { "code": 200, "data": 456 }  // 返回章节 ID

步骤 4: 上传课时视频
POST /pc/course/section/video/upload?courseId=123&sectionId=456
FormData: file=[视频文件]
Response: { "code": 200, "data": {...} }

步骤 5: 上传课程资料
POST /pc/course/material/upload?courseId=123&materialName=课后习题
FormData: file=[ZIP 压缩包]
Response: { "code": 200, "data": 789 }  // 返回资料 ID
```

## 八、注意事项

1. **文件大小限制**:必须在枚举类中明确定义，避免硬编码
2. **文件类型校验**:由 `FileUploadUtils.assertAllowed()` 统一处理
3. **权限控制**:建议使用 `@PreAuthorize` 注解 + 自定义权限表达式
4. **并发上传**:同一课程的封面上传需要加锁，防止覆盖
5. **文件删除**:课程删除时应同步删除关联文件（定时任务或事件驱动）
6. **CDN 加速**:建议生产环境使用 CDN 加速视频和资料下载
