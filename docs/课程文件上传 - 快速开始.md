## 二、调用方式（两种场景）

### 场景 A：前端直接上传封面文件（推荐）

#### Controller 层（需要修改）

```java
@Anonymous
@Log(title = "课程", businessType = BusinessType.INSERT)
@ApiOperation("新增课程")
@PostMapping(value = "/create", consumes = MediaType.MULTIPART_FORM_DATA_VALUE)
public R<Long> createCourse(
        @RequestPart("courseCreateDTO") String courseCreateDTOJson,
        @RequestPart(value = "coverFile", required = false) MultipartFile coverFile) {
    try {
        // 1. 解析 JSON 为 DTO
        CourseCreateDTO dto = JSON.parseObject(courseCreateDTOJson, CourseCreateDTO.class);
        
        // 2. 设置封面文件
        if (coverFile != null && !coverFile.isEmpty()) {
            dto.setCoverFile(coverFile);
        }
        
        // 3. 调用 Service 创建课程
        Long userId = getUserId();
        Long courseId = courseManageService.createCourseWithSections(dto, userId);
        
        return R.ok(courseId);
    } catch (Exception e) {
        log.error("创建课程失败：{}", e.getMessage(), e);
        return R.fail("创建失败：" + e.getMessage());
    }
}
```

#### 前端请求示例

```javascript
// 使用 FormData 提交
const formData = new FormData();

// 1. 添加课程信息（JSON 字符串）
formData.append('courseCreateDTO', JSON.stringify({
    title: "Java 高级编程实战",
    type: "media",
    price: 99.00,
    tPrice: 199.00,
    intro: "从入门到精通",
    status: 1,
    tagIds: [1, 2, 3],
    sections: [/* 章节数据 */]
}));

// 2. 添加封面图片
const coverInput = document.querySelector('input[type="file"]');
if (coverInput.files.length > 0) {
    formData.append('coverFile', coverInput.files[0]);
}

// 3. 发送请求
axios.post('/pc/course/create', formData, {
    headers: {
        'Content-Type': 'multipart/form-data',
        'appid': 'your-app-id'
    }
});
```

### 场景 B：先上传图片，再创建课程（分离式）

#### 步骤 1：调用通用上传接口

```javascript
// 1. 上传封面图片
const coverFormData = new FormData();
coverFormData.append('file', coverFile);

axios.post('/common/upload', coverFormData).then(response => {
    const coverUrl = response.data.url; // 获取上传后的 URL
    
    // 2. 使用 URL 创建课程
    createCourse(coverUrl);
});
```

#### 步骤 2：创建课程

```javascript
function createCourse(coverUrl) {
    const courseData = {
        title: "Java 高级编程实战",
        cover: coverUrl, // 使用已上传的 URL
        type: "media",
        price: 99.00,
        // ... 其他字段
    };
    
    axios.post('/pc/course/create', courseData, {
        headers: {
            'Content-Type': 'application/json',
            'appid': 'your-app-id'
        }
    });
}
```

**注意**: 此方式不需要修改 Controller，使用现有的 `@RequestBody` 注解即可。

## 三、文件上传流程图解

```
用户发起请求
    ↓
Controller 接收（MultipartFile）
    ↓
DTO.setCoverFile()
    ↓
Service.createCourseWithSections()
    ↓
判断是否有封面文件？
    ├─ 否 → 使用原有 cover URL
    └─ 是 → uploadCourseCover()
              ↓
         FileUploadUtils.upload(file, COURSE_COVER)
              ↓
         1. 校验文件名长度（≤100 字符）
         2. 校验文件大小（≤5MB）
         3. 校验文件类型（bmp/gif/jpg/jpeg/png）
         4. 生成 UUID 文件名
         5. 保存到物理路径
         6. 返回访问 URL
              ↓
         course.setCover(上传后的 URL)
              ↓
courseMapper.insertCourse(course)
              ↓
保存章节、标签等
              ↓
返回课程 ID
```

## 四、文件存储说明

### 存储路径结构

```
本地文件系统:
{profile}/course/cover/2026/03/31/{UUID}.jpg
    ├─ profile: 配置文件中指定的根目录（如 D:\ruoyi\upload）
    ├─ course/cover: 课程封面专用目录
    ├─ 2026/03/31: 日期分目录
    └─ {UUID}.jpg: UUID 命名的文件

访问 URL:
http://localhost:8080/profile/course/cover/2026/03/31/{UUID}.jpg
```

### 配置文件位置

在 `application.yml` 中配置：

```yaml
ruoyi:
  profile: D:/ruoyi/upload  # Windows 示例
  # profile: /home/ruoyi/upload  # Linux 示例
```

## 六、完整代码示例

### 6.1 Service 层核心代码

```java
@Override
@Transactional(rollbackFor = Exception.class)
public Long createCourseWithSections(CourseCreateDTO courseCreateDTO, Long userId) {
    // 1. 基础校验
    if (StringUtils.isEmpty(courseCreateDTO.getTitle())) {
        throw new ServiceException("课程标题不能为空");
    }
    
    // 2. 处理封面上传
    String coverUrl = courseCreateDTO.getCover();
    if (courseCreateDTO.getCoverFile() != null && !courseCreateDTO.getCoverFile().isEmpty()) {
        try {
            coverUrl = uploadCourseCover(courseCreateDTO.getCoverFile());
        } catch (FileSizeLimitExceededException | FileNameLengthLimitExceededException e) {
            throw new ServiceException("封面图片大小或文件名不符合要求");
        } catch (InvalidExtensionException e) {
            throw new ServiceException("封面图片仅支持 bmp、gif、jpg、jpeg、png 格式");
        } catch (IOException e) {
            log.error("上传封面图片失败：{}", e.getMessage(), e);
            throw new ServiceException("上传封面图片失败：" + e.getMessage());
        }
    }
    
    // 3. 保存课程
    OshCoures course = new OshCoures();
    BeanUtils.copyProperties(courseCreateDTO, course);
    course.setCover(coverUrl); // 关键：设置上传后的路径
    course.setCreateBy(String.valueOf(userId));
    course.setCreateTime(DateUtils.getNowDate());
    course.setUpdateTime(DateUtils.getNowDate());

    int result = courseMapper.insertCourse(course);
    if (result <= 0) {
        throw new ServiceException("新增课程失败");
    }

    // 4-6. 保存章节、更新数量、保存标签...
    saveSectionsAndMaterials(course.getId(), courseCreateDTO.getSections());
    saveCourseTags(course.getId(), courseCreateDTO.getTagIds());
    
    return course.getId();
}

/**
 * 上传课程封面（私有方法）
 */
private String uploadCourseCover(MultipartFile coverFile) 
        throws IOException, FileSizeLimitExceededException, 
               FileNameLengthLimitExceededException, InvalidExtensionException {
    return FileUploadUtils.upload(coverFile, UploadPathEnum.COURSE_COVER);
}
```

## 八、常见问题 FAQ

### Q1: 为什么要在 DTO 中添加 `transient` 修饰符？

**A**: `transient` 防止文件对象被序列化。`MultipartFile` 包含流对象，无法通过网络传输，只能在当前请求中使用。

### Q2: 如果已经创建了课程，想替换封面怎么办？

**A**: 有两种方案：
1. 调用修改课程接口，传递新的 `coverFile`
2. 单独提供封面上传接口，返回 URL 后调用课程修改接口

### Q3: 上传失败后，已上传的文件会清理吗？

**A**: 当前版本不会自动清理。建议：
- 添加定时任务清理孤立文件
- 或在事务失败时手动删除（需要在 catch 块中处理）

### Q4: 可以一次上传多个文件吗？

**A**: 可以。参考 `COURSE_MATERIAL` 枚举，在 DTO 中添加 `List<MultipartFile> materialFiles` 字段，循环调用 `FileUploadUtils.upload()` 即可。

### Q5: 如何集成阿里云 OSS？

**A**: 修改 `FileUploadUtils.upload()` 方法，根据配置决定使用本地存储还是 OSS：

```java
public static final String upload(MultipartFile file, UploadPathEnum pathEnum) {
    if ("oss".equals(RuoYiConfig.getStorageType())) {
        return OssUtil.upload(file, pathEnum.getRelativePath());
    } else {
        // 现有本地存储逻辑
    }
}
```
